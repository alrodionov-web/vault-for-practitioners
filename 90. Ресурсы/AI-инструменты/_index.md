# AI-инструменты для практики

## Три сценария использования AI

### 1. **Agent** — технические задачи
Установка ПО, отладка, настройка окружения, решение технических проблем.

**Приоритеты:** точность, техническая грамотность, безопасность, стоимость.

### 2. **Think** — аналитика и стратегия
Размышления о позиционировании, анализ клиентских кейсов, стратегическое планирование, рефлексия.

**Приоритеты:** глубина анализа, системность, креативность, практическая ценность.

### 3. **Text** — создание контента
Написание постов для соцсетей, статей, email-рассылок, маркетинговых материалов.

**Приоритеты:** качество текста, стиль, вовлечение аудитории, стоимость производства.

## Детальные рекомендации

### Для начинающей практики (бюджет до $10/мес)
- **Основная модель:** DeepSeek V3.2 — для всех задач
- **Для контента:** GPT-4o mini — массовое производство
- **Для анализа:** Claude 3 Haiku эпизодически
- **ROI:** максимальный, качество 8.5/10

### Для растущей практики (бюджет $20-50/мес)
- **Основная:** Claude 3 Haiku — 80% задач
- **Критические задачи:** GPT-4o — 15%
- **Русский контент:** YandexGPT 3 — 5%
- **ROI:** отличный, качество 9.0/10

### Для устоявшейся практики (бюджет $100-200/мес)

- **Стратегия:** Claude 3.5 Sonnet — 20%

- **ROI:** премиум, качество 9.5/10

## Практические советы

### Экономия токенов:
1. Четкие системные промпты
2. Только необходимый контекст
3. Итеративный подход (короткие запросы + уточнения)
4. Кэширование похожих ответов

### Качество промптов:
- **Agent:** "Шаг за шаг", "Проверь каждый шаг", "Что может пойти не так?"
- **Think:** "Проанализируй с трёх сторон", "Какие слепые зоны?", "Системные последствия"
- **Text:** "Тон: [описание]", "ЦА: [описание]", "Избегай: [список]", "Включи: [элементы]"

### Мониторинг расходов:
1. Дневные/месячные лимиты
2. Разные API ключи для разных сценариев
3. Логирование использования
4. Квартальный пересмотр ROI

## Тестирование моделей

### Для точного выбора:
1. Возьмите 3-5 типовых задач каждого сценария
2. Запустите на 2-3 моделях-кандидатах
3. Оцените результаты по критериям:
   - **Agent:** точность команд, безопасность, объяснения
   - **Think:** глубина, практичность, структурированность
   - **Text:** грамотность, стиль, вовлечение
4. Рассчитайте реальный ROI для вашего использования
