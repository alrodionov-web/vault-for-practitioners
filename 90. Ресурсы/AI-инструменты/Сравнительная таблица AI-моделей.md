## Практические советы

### Экономия токенов
1. **Системные промпты** - четкие инструкции экономят output токены
2. **Структурированный контекст** - только необходимые детали
3. **Итеративный подход** - короткие запросы с уточнениями
4. **Кэширование** - повторное использование похожих ответов

### Качество промптов
1. **Для Agent:** "Шаг за шаг", "Проверь каждый шаг", "Что может пойти не так?"
2. **Для Think:** "Проанализируй с трех сторон", "Какие слепые зоны?", "Системные последствия"
3. **Для Text:** "Тон: [описание]", "ЦА: [описание]", "Избегай: [список]", "Включи: [элементы]"

### Мониторинг расходов
1. Установить дневные/месячные лимиты
2. Отдельные API ключи для разных сценариев
3. Логирование использования по типам задач
4. Регулярный пересмотр ROI (раз в квартал)
