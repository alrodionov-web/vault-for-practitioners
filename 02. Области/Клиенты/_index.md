# Клиенты

CRM вашей практики. Каждый клиент - отдельная папка.

## CRITICAL: Конфиденциальность

- Используйте **псевдонимы или инициалы**, не реальные имена
- Не записывайте телефоны, адреса, паспортные данные
- Заметки хранятся только на вашем компьютере

## Структура

```
Клиенты/
├── _index.md           # Этот файл
├── _шаблон-сессии.md   # Шаблон для новой заметки
├── Клиент А/
│   ├── Профиль.md      # Запрос, план работы, контекст
│   └── YYYY-MM-DD.md   # Заметки по сессиям
└── ...
```
