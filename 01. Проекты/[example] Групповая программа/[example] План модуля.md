# Модуль 1: Знакомство с собой

## Параметры

- Длительность: 3 часа
- Формат: онлайн, Zoom
- Участники: 8-12 человек

## Программа

### Часть 1: Открытие (45 мин)
- Представление и знакомство (30 мин)
- Правила группы: конфиденциальность, уважение (15 мин)

### Часть 2: Основная работа (90 мин)
- Мини-лекция (20 мин)
- Упражнение в парах (30 мин)
- Групповое обсуждение (20 мин)
- Перерыв (10 мин)
- Индивидуальное упражнение (10 мин)

### Часть 3: Интеграция (45 мин)
- Шеринг по кругу (30 мин)
- Домашнее задание (5 мин)
- Закрытие (10 мин)
