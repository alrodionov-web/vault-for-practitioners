# Проекты

Каждый проект - отдельная папка с `_index.md` внутри.

**Проект** = деятельность с конкретной целью и сроком. Когда цель достигнута - переместить в `04. Архив/`.

## Структура проекта

```
Название проекта/
├── _index.md        # Описание, цели, статус
└── ...              # Материалы проекта
```

Шаблон для нового проекта: `03. Ресурсы/Шаблоны/template-project.md`
