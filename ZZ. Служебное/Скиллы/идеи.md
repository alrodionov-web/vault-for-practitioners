# Скилл: Идеи

## CRITICAL: Правила (прочитай перед выполнением)

- Путь к файлу идеи: `04. Идеи/YYYY-MM-DD [Название].md`
  - НЕПРАВИЛЬНО: `04. Идеи/Название.md` (без даты)
  - ПРАВИЛЬНО: `04. Идеи/2026-04-14 Групповой марафон.md`
- Один вопрос за раз - не перегружать
- 2-3 подхода - всегда предлагать альтернативы
- Без оценки - не говорить "отличная идея" или "это не сработает"
- НЕ сохранять без подтверждения

---

## Процесс

### Фаза 1. Понять идею

1. Спросить: "Расскажите идею в двух-трех предложениях. Не нужно подробностей - просто суть."
2. Задавать уточняющие вопросы **по одному**:
   - Предпочитать вопросы с вариантами (a/b/c), а не открытые
   - Фокус: для кого, какую проблему решает, почему сейчас
   - Не больше 5-7 вопросов на эту фазу

### Фаза 2. Предложить подходы

1. Предложить 2-3 разных подхода к реализации
2. Для каждого: краткое описание, плюсы, минусы
3. Дать рекомендацию с обоснованием

### CRITICAL: СТОП. Спросить:

> "Какой подход ближе? Или хотите комбинацию?"

Ждать ответа.

### Фаза 3. Развить выбранный подход

Раскрыть подробнее:
- Что конкретно делаем
- Для кого
- В каком формате
- Какой результат для клиента/участника
- Первые шаги

Показывать по частям, каждую подтверждать.

### CRITICAL: СТОП. Спросить:

> "Это то, что вы имели в виду? Что поправить?"

Ждать ответа.

### Фаза 4. Зафиксировать

Собрать итоговый документ:

```
# [Название идеи]
Дата: YYYY-MM-DD

## Суть
[1-2 предложения]

## Для кого
[целевая аудитория]

## Проблема / потребность
[что решаем]

## Подход
[выбранный вариант]

## Формат и детали
[конкретика]

## Результат
[что получит клиент/участник]

## Первые шаги
1. [шаг]
2. [шаг]
3. [шаг]

## Открытые вопросы
- [что еще решить]
```

### CRITICAL: СТОП. Показать документ и спросить:

> "Сохраняем? Хотите что-то поправить?"

Только после подтверждения:
1. Сохранить в `04. Идеи/YYYY-MM-DD [Название].md`
   - НЕПРАВИЛЬНО: `04. Идеи/Название.md`
   - ПРАВИЛЬНО: `04. Идеи/2026-04-14 Групповой марафон.md`
2. Предложить: "Хотите сразу создать проект в `22. Проекты/` или пока оставим как идею?"
