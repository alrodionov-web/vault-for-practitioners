# Скилл: Стратегия

## CRITICAL: Правила (прочитай перед выполнением)

- Цели сохранять ТОЛЬКО в `01. Планирование/Цели.md`
- НЕ ставить цели за пользователя - помогать формулировать
- PEPE-проверка молча, вопросы только по проблемным пунктам
- Каждую цель подтверждать отдельно
- `[PLACEHOLDER]` если пользователь не может назвать число

## Процесс

### Шаг 1. Проверить текущее состояние

1. Прочитать `01. Планирование/Цели.md`
2. Если цели уже есть -> предложить: "У вас есть цели. Хотите пересмотреть или скорректировать?"
3. Если целей нет -> перейти к шагу 2

### Шаг 2. Выбрать подход

### CRITICAL: СТОП. Показать два варианта и ждать выбора:

> Есть два пути. Какой ближе?
>
> 1. **"У меня есть цели, помоги оформить"** - вы расскажете, к чему стремитесь, а я помогу сформулировать и оцифровать
> 2. **"Я пока не знаю куда иду, давай разберемся"** - я посмотрю чем вы занимались, и мы от этого оттолкнемся

НЕ переходить дальше без выбора пользователя.

### Шаг 3A. Путь "от целей"

1. Спросить: "Какие 3-5 направлений важны на ближайший год?" Примеры: практика, доход, обучение, здоровье, контент, личная жизнь
2. По каждому направлению попросить сформулировать цель своими словами
3. Проверить каждую цель по PEPE самостоятельно. Задавать вопросы ТОЛЬКО по проблемным пунктам:
   - Отрицание: "'Перестать бояться продавать' - с отрицанием. А когда страха нет, вы что делаете?"
   - Внутренняя: "'Понять свою нишу' - внутренняя. Когда поняли - что изменится снаружи?"
   - Чужая: "Это вы хотите, или от вас этого ждут?"
   - Не зажигает: "Что в этой цели такого, что стоит заниматься?"
4. Оцифровать - определить тип:
   - "10 клиентов в месяц", "3 поста в неделю" -> **постоянная** (метрика). Уточнить текущее значение.
   - "Запустить группу до сентября" -> **разовая** (результат). Уточнить срок.
   - Если неясно, спросить: "Это постоянная цель (держать каждый месяц) или разовая (сделать и закрыть)?"

### CRITICAL: СТОП. Показать итоговую таблицу всех целей:

> "Вот ваши цели. Все верно?"

Ждать подтверждения.

### Шаг 3B. Путь "от активности"

1. Прочитать:
   - Обзоры месяца из `01. Планирование/` (если есть)
   - Недельные планы (`Неделя NN.md`)
   - Логи сессий из `ZZ. Служебное/sessions/`
2. Показать: "За последнее время вы занимались X, Y, Z. Больше всего времени ушло на A и B."
3. Спросить: "Хотите поставить цели в этих направлениях? Или есть что-то, чего здесь не видно, но что важно?"
4. Дальше как в шаге 3A: формулировка -> PEPE -> оцифровка -> подтверждение

### Шаг 4. Сохранение

Только после подтверждения:
1. Обновить `01. Планирование/Цели.md`
   - НЕПРАВИЛЬНО: создать новый файл `Цели 2026.md`
   - ПРАВИЛЬНО: обновить существующий `01. Планирование/Цели.md`
2. Обновить `ZZ. Служебное/текущий-фокус.md`

## Формат целей в файле

```
## Направления на год

1. [Направление]
2. [Направление]
3. ...

## Цели

### [Направление]

**[Формулировка цели]**
- Тип: постоянная / разовая
- Метрика: [что измеряем] | Текущее: [N] | Цель: [M]  <- для постоянных
- Результат: [что получим] | Срок: [дата]  <- для разовых
- Статус: в работе / достигнута / пауза
```
