# Скилл: Транскрибация

Расшифровка аудио и видео файлов в текст. Технический скилл.

## Роутинг по намерению

- "расшифруй", "транскрибируй", "переведи в текст" -> Расшифровка файла
- "расшифруй видео с YouTube", "ютуб" -> Расшифровка YouTube

## Требования

- Python 3.10+ установлен
- Виртуальное окружение: `ZZ. Служебное/venv/`
- Зависимости: `pip install mlx-whisper` (для Mac с Apple Silicon)

## Расшифровка файла

1. Спросить путь к файлу (или попросить перетащить в окно)
2. Определить формат (mp3, mp4, m4a, wav, webm)
3. Активировать venv и запустить:

```bash
source "ZZ. Служебное/venv/.venv/bin/activate"
HF_HOME="/tmp/huggingface_$(whoami)" python3 -c "
import mlx_whisper
result = mlx_whisper.transcribe('[путь к файлу]', path_or_hf_repo='mlx-community/whisper-large-v3-turbo')
print(result['text'])
"
```

4. Спросить: "Куда сохранить результат?"
   - В папку клиента -> `20. Клиенты/[Псевдоним]/`
   - В проект -> `22. Проекты/[Название]/`
   - В ресурсы -> `90. Ресурсы/`
   - Просто показать (не сохранять)
5. Сохранить как `[YYYY-MM-DD] Расшифровка [название].md`

## Расшифровка YouTube

1. Спросить URL видео
2. Скачать аудио:

```bash
source "ZZ. Служебное/venv/.venv/bin/activate"
pip install -q yt-dlp
yt-dlp -x --audio-format mp3 -o "/tmp/yt_audio.mp3" "[URL]"
```

3. Транскрибировать как обычный файл (см. выше)
4. Удалить временный файл

## Правила

- Всегда использовать модель `large-v3-turbo`
- Предупредить пользователя: первый запуск скачивает модель (~1.5 GB)
- Если Mac без Apple Silicon - предложить альтернативу (openai-whisper)
- Результат показать пользователю перед сохранением
- Если расшифровка сессии с клиентом - напомнить про конфиденциальность
