# Скилл: Маркетинг

Мета-скилл: личный бренд, написание постов, упаковка продуктов и услуг.

## Роутинг по намерению

- "напиши пост", "пост в телеграм", "контент" -> Написать пост
- "опиши продукт", "упакуй услугу", "JTBD" -> Упаковка продукта
- "мой стиль", "тон коммуникации", "как я пишу" -> Определить стиль
- "контент-план", "что писать", "темы для постов" -> Контент-план
- "переделай пост", "адаптируй для VK/Instagram" -> Репурпозинг

## Подкоманды

### Написать пост

1. Спросить: "О чем хотите написать? Тема, мысль, кейс, наблюдение."
2. Прочитать `10. Маркетинг/Личный бренд/Мой бренд.md` (если есть) - контекст позиционирования
3. Прочитать `10. Маркетинг/Личный бренд/Мой стиль.md` (если есть)
4. Прочитать `10. Маркетинг/Личный бренд/Контент-план.md` (если есть)
4. Предложить 2-3 варианта структуры (короткий / средний / длинный)
5. Написать черновик в выбранном формате
6. Показать и спросить: "Так? Поправить?"
7. После одобрения сохранить в `10. Маркетинг/Личный бренд/Черновики/YYYY-MM-DD Тема.md`

Правила:
- НИКОГДА не использовать реальные имена клиентов
- Если кейс - анонимизировать до неузнаваемости
- Писать голосом пользователя (из "Мой стиль.md"), не generic
- Не морализировать и не поучать
- Длина по умолчанию: 1000-1500 знаков (Telegram)

### Упаковка продукта (JTBD)

1. Спросить: какой продукт/услугу описываем
2. Прочитать существующие описания в `10. Маркетинг/Продукты и услуги/`
3. Пройти по вопросам:
   - Какую работу "нанимает" клиент? Что пытается решить?
   - В какой ситуации возникает потребность?
   - Как клиент решает это сейчас?
   - Почему текущие решения не работают?
   - Что изменится после работы с вами?
4. Сформулировать описание:
   - Одно предложение: для кого и зачем
   - Работа клиента (JTBD)
   - Как вы выполняете эту работу
   - Результат
   - Параметры (формат, длительность, цена)
5. Сохранить в `10. Маркетинг/Продукты и услуги/[название].md`

Правила:
- Описание от лица клиента ("вы хотите..."), не от лица практика
- Конкретные результаты, не абстрактные обещания
- Если цена неизвестна - `[PLACEHOLDER]`

### Определить стиль

1. Если `Мой стиль.md` не существует - провести интервью:
   - "Как бы вы описали свой тон? (теплый, строгий, ироничный, научный...)"
   - "Какие слова вы часто используете?"
   - "Чего вы точно НЕ хотите в своих текстах?"
   - "Кто ваша аудитория? Как вы к ней обращаетесь?"
   - "Покажите 2-3 поста, которые вам нравятся по стилю"
2. Сформировать `Мой стиль.md`:
   - Тон и голос
   - Словарь (используем / избегаем)
   - Обращение к аудитории
   - Табу
   - Примеры хороших формулировок
3. Если файл есть - предложить обновить

### Контент-план

1. Прочитать `Мой стиль.md` и `Продукты и услуги/`
2. Спросить: "На какой период? Неделя / месяц?"
3. Спросить: "Какие темы сейчас актуальны? Есть ли события, запуски, даты?"
4. Предложить план: дата - тема - формат (пост/сторис/статья) - платформа
5. Сохранить/обновить `10. Маркетинг/Личный бренд/Контент-план.md`

### Репурпозинг

1. Спросить: какой контент адаптируем и для какой платформы
2. Прочитать исходный файл
3. Адаптировать: длина, формат, хештеги, CTA
4. Показать результат
