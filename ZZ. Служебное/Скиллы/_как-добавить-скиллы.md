# Как добавить скиллы в Zed

Скиллы - это типовые запросы, которые AI выполняет по заранее написанному протоколу. Каждый скилл - это текст, который нужно один раз добавить в Zed Rules Library.

## Инструкция

1. Откройте Zed
2. Откройте Rules Library: `Cmd+Alt+L` (или `Cmd+Shift+P` -> "Agent: Open Rules Library")
3. Нажмите "+" чтобы создать новое правило
4. Назовите правило (например, "клиенты")
5. Скопируйте текст из соответствующего файла в этой папке
6. Сохраните

## Как использовать

Вы можете просто описать что хотите - AI определит нужный скилл автоматически:
- "запиши сессию с клиентом А" -> скилл Клиенты
- "спланируй неделю" -> скилл Планирование
- "напиши пост про выгорание" -> скилл Маркетинг
- "расшифруй запись" -> скилл Транскрибация

## Доступные скиллы

### Основные

| Файл | Что делает |
|------|-----------|
| `клиенты.md` | Профиль клиента, запись сессии, динамика, подготовка к супервизии |
| `маркетинг.md` | Посты (TG/VK), стиль коммуникации, упаковка услуг (JTBD), контент-план |
| `финансы.md` | Учет сессий и оплат, месячный отчет, подготовка к налогам |
| `здоровье.md` | Чек выгорания, рефлексия, баланс нагрузки |
| `группы.md` | Программа тренинга/марафона, план занятия, участники |
| `идеи.md` | Развитие идей через структурированный диалог |

### Планирование

| Файл | Что делает |
|------|-----------|
| `ежедневное-планирование.md` | Утренний статус + вечерняя рефлексия |
| `недельное-планирование.md` | Задачи, приоритеты, раскидка по дням |
| `обзор-месяца.md` | Факт vs план, прогресс по целям |
| `стратегия.md` | Цели на год по модели PEPE |

### Технические

| Файл | Что делает |
|------|-----------|
| `calendar.md` | Расписание дня из календаря (ICS) |
| `транскрибация.md` | Расшифровка аудио/видео (mlx-whisper) |

## Совет

Можно пометить часто используемые скиллы как "default" (иконка скрепки) - тогда они будут автоматически включаться в каждый разговор.
