# Скилл: Финансы

Учет сессий и дохода для самозанятого/ИП практика.

## Роутинг по намерению

- "запиши оплату", "получил оплату", "клиент заплатил" -> Записать оплату
- "сколько сессий", "статистика сессий" -> Подсчет сессий
- "отчет за месяц", "финансовый отчет", "сколько заработал" -> Месячный отчет
- "подготовь к налогам", "для бухгалтера", "самозанятый отчет" -> Налоговая подготовка

## Структура данных

Годовой лог: `02. Финансы/YYYY.md`

```
# Финансы YYYY

## Настройки
- Стоимость индивидуальной сессии: [PLACEHOLDER] руб
- Стоимость групповой сессии: [PLACEHOLDER] руб
- Статус: самозанятый / ИП / физлицо

## Журнал

| Дата | Клиент | Тип | Сумма | Способ | Статус |
|------|--------|-----|-------|--------|--------|
| 2026-01-15 | Клиент А | инд | 5000 | перевод | оплачено |
```

## Подкоманды

### Записать оплату

1. Спросить: кто, когда, сумма, способ оплаты (перевод/наличные/карта)
2. Если клиент не указан - спросить
3. Если сумма не указана - подставить из "Настройки"
4. Добавить строку в таблицу `02. Финансы/YYYY.md`
5. Показать: "Записано: [клиент], [сумма], [дата]"

### Подсчет сессий

1. Прочитать `02. Финансы/YYYY.md`
2. Спросить: "За какой период? Неделя / месяц / квартал / весь год?"
3. Показать:
   - Всего сессий: N
   - По типам: индивидуальных N, групповых N
   - По клиентам: топ-5
   - Сравнение с предыдущим периодом (если есть данные)

### Месячный отчет

1. Прочитать `02. Финансы/YYYY.md` за нужный месяц
2. Сформировать отчет:

```
# Отчет за [месяц YYYY]

## Доход
- Всего: N руб
- Индивидуальные сессии: N x [цена] = N руб
- Групповые: N x [цена] = N руб

## Сессии
- Всего проведено: N
- Отмены/переносы: N

## Сравнение
- Прошлый месяц: N руб (разница: +/-N%)

## Неоплаченное
- [если есть записи со статусом "ожидание"]
```

3. Сохранить в `02. Финансы/отчеты/YYYY-MM Отчет.md`

### Налоговая подготовка

1. Прочитать все записи за год из `02. Финансы/YYYY.md`
2. Сформировать сводку:
   - Общий доход за год
   - Разбивка по кварталам
   - Разбивка по месяцам
   - Количество клиентов
3. Если самозанятый: напомнить про лимит 2.4 млн/год, показать сколько осталось
4. Сохранить в `02. Финансы/отчеты/YYYY Годовая сводка.md`

## Правила

- Имена клиентов - только псевдонимы (как в `20. Клиенты/`)
- Суммы и финансовые данные хранятся локально, не отправляются вовне
- `[PLACEHOLDER]` для неизвестных значений
- Не давать налоговых консультаций - только структурировать данные
