## Первое подключение

При первом запросе к планированию или календарю агент спрашивает:

> В каком сервисе вы ведете календарь?

> 1. Google Calendar
> 2. Outlook / Microsoft 365
> 3. Яндекс Календарь
> 4. Apple Calendar (iCloud)
> 5. Другой / нет календаря

Затем:

> Как подключить?
>
> a. **Полный доступ (CalDAV)** - читать и добавлять события. Нужен пароль приложения.
> b. **Только чтение (ICS)** - видеть расписание, но не добавлять события.
> c. **Без календаря** - диктовать встречи текстом.

## Инструкции по подключению

### CalDAV (полный доступ)

**Яндекс Календарь:**
1. Перейти на https://id.yandex.ru/security/app-passwords
2. Создать пароль приложения для "Календарь" (или "Другое")
3. Скопировать пароль и вставить агенту
4. CalDAV URL: `https://caldav.yandex.ru/calendars/[login]@yandex.ru/`

**Google Calendar:**
1. Включить двухфакторную аутентификацию в Google
2. Перейти на https://myaccount.google.com/apppasswords
3. Создать пароль приложения
4. CalDAV URL: `https://www.googleapis.com/caldav/v2/calid/events`

**Apple Calendar (iCloud):**
1. Перейти на https://appleid.apple.com
2. Создать пароль приложения
3. CalDAV URL: `https://caldav.icloud.com/`

**Outlook / Microsoft 365:**
- CalDAV не поддерживается напрямую, использовать ICS

### ICS (только чтение)

**Google Calendar:**
1. Откройте calendar.google.com
2. Справа от нужного календаря: три точки -> Настройки
3. Прокрутите до "Интеграция календаря"
4. Скопируйте "Секретный адрес в формате iCal"

**Outlook / Microsoft 365:**
1. Откройте outlook.com -> Настройки (шестеренка)
2. Календарь -> Общие календари
3. "Опубликовать календарь" -> "Можно просматривать все сведения"
4. Скопируйте ссылку ICS

**Яндекс Календарь:**
1. Наведите на название календаря -> значок настроек
2. Вкладка "Экспорт"
3. Скопируйте ссылку iCal

**Apple Calendar (iCloud):**
1. В Calendar на Mac: наведите на календарь -> Info (i)
2. Поставьте галочку "Публичный календарь"
3. Скопируйте ссылку
