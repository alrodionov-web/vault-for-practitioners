# Как добавить скиллы в Zed

Скиллы - это типовые запросы, которые AI выполняет по заранее написанному протоколу. Каждый скилл - это текст, который нужно один раз добавить в Zed Rules Library.

## Инструкция

1. Откройте Zed
2. Откройте Rules Library: `Cmd+Alt+L` (или `Cmd+Shift+P` -> "Agent: Open Rules Library")
3. Нажмите "+" чтобы создать новое правило
4. Назовите правило (например, "личный-бренд")
5. Скопируйте текст из соответствующего файла в этой папке
6. Сохраните

## Как использовать

В Agent Panel напишите `@имя-скилла` и AI выполнит протокол. Например:

- `@личный-бренд` - помочь написать пост
- `@jtbd-описание-продукта` - описать продукт
- `@недельное-планирование` - спланировать неделю
- `@ежедневное-планирование` - план на день
- `@здоровье` - проверка состояния

## Доступные скиллы

| Файл | Имя в Rules Library | Что делает |
|------|-------------------|-----------|
| `личный-бренд.md` | личный-бренд | Пост из мысли/кейса |
| `jtbd-описание-продукта.md` | jtbd-описание-продукта | Описание продукта через JTBD |
| `недельное-планирование.md` | недельное-планирование | Обзор + приоритеты + план |
| `ежедневное-планирование.md` | ежедневное-планирование | Утренний план + вечерняя рефлексия |
| `здоровье.md` | здоровье | Чек выгорания, рефлексия |

## Совет

Можно пометить часто используемые скиллы как "default" (иконка скрепки) - тогда они будут автоматически включаться в каждый разговор.
