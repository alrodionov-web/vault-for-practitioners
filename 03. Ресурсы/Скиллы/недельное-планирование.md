# Скилл: Недельное планирование

Еженедельная сессия планирования (30-45 мин, утро понедельника).

## Процесс

### Фаза 1: Сбор картины (5 мин)

1. Прочитай `ZZ. Служебное/текущий-фокус.md` - что было в работе
2. Прочитай последнюю заметку из `00. Планирование/` - план прошлой недели
3. Покажи короткую сводку:

```
## Ретро прошлой недели
- Запланировано: N задач, сделано: M
- Не сделано: [список]

## Текущее состояние областей
- Маркетинг: [когда последний пост, что в контент-плане]
- Клиенты: [сколько активных, сколько сессий на неделе]
- Здоровье: [есть ли признаки перегрузки]
```

### Фаза 2: Приоритизация (15-20 мин)

Пройди по каждой области:

1. **Клиенты:** Расписание сессий на неделю. Есть ли новые обращения?
2. **Маркетинг:** Что публикуем? Есть ли черновики?
3. **Здоровье:** Как состояние? Нагрузка в норме?
4. **Проекты:** Есть ли активные проекты с дедлайнами?
5. **Из головы:** Что еще нужно сделать, чего нет в системе?

Спроси: что must, что should, что nice-to-have?

### Фаза 3: План по дням (10 мин)

1. Предложи распределение задач по дням
2. Учитывай: дни с сессиями - менее продуктивны для другой работы
3. Не планируй больше 3-4 задач в день помимо сессий

### Фаза 4: Фиксация (5 мин)

1. Создай `00. Планирование/YYYY-MM-DD Недельный план.md`
2. Обнови `ZZ. Служебное/текущий-фокус.md`

## Формат выходной заметки

```
# Недельный план YYYY-MM-DD

## Ключевые задачи недели
1. [Must] Задача (~Xч)
2. [Should] Задача (~Xч)

## План по дням

### Понедельник
- [ ] Задача
- Сессии: [список]

### Вторник
...

## Заметки
[мысли, решения]
```
