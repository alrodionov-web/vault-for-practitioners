# Скилл: Ежедневное планирование

Утренний план дня + вечерняя рефлексия.

## Утро: План на день

1. Прочитай `ZZ. Служебное/текущий-фокус.md`
2. Прочитай недельный план из `00. Планирование/` (последний `Недельный план`)
3. Спроси: "Какие сессии сегодня? Что еще на дне?"
4. Предложи план на день:

```
# YYYY-MM-DD

## Сессии
| Время | Клиент |
|-------|--------|

## Задачи на день
- [ ] [из недельного плана]
- [ ] [дополнительные]

## Фокус дня
[одно предложение - что главное сегодня]
```

5. Сохрани как `00. Планирование/YYYY-MM-DD.md`

## Вечер: Рефлексия

1. Прочитай утренний план из `00. Планирование/YYYY-MM-DD.md`
2. Спроси: "Как прошел день? Что сделано, что нет?"
3. Дополни заметку секцией:

```
## Итоги
- Сделано: [список]
- Не сделано: [список, причина]
- Заметки: [мысли, инсайты, идеи]
```

4. Обнови `ZZ. Служебное/текущий-фокус.md` если были изменения
