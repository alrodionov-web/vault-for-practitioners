# Хранилище помогающего практика

Шаблон персонального хранилища знаний для психологов, коучей, тренеров и консультантов. Работает в редакторе Zed с AI-ассистентом.

## Быстрый старт

### 1. Установить Zed

Скачать с [zed.dev](https://zed.dev). Бесплатный редактор с встроенным AI-ассистентом.

### 2. Подключить API-ключ

Zed поддерживает любые LLM-провайдеры. Нужен ключ хотя бы одного:

| Провайдер | Где получить | Модель |
|-----------|-------------|--------|
| Anthropic (Claude) | [console.anthropic.com](https://console.anthropic.com) | Claude Sonnet - рекомендуется |
| OpenAI (ChatGPT) | [platform.openai.com](https://platform.openai.com) | GPT-4o |
| Google (Gemini) | [aistudio.google.com](https://aistudio.google.com) | Gemini Pro |

**Как вставить ключ:** Открыть Zed -> Agent Panel (`Cmd+Shift+P` -> "Agent Panel") -> настройки -> выбрать провайдер -> вставить ключ. Ключ хранится в системной связке ключей.

### 3. Открыть хранилище

1. Скопировать эту папку в удобное место (например, `~/Documents/Моя практика/`)
2. Открыть папку в Zed: `File -> Open Folder`
3. Открыть Agent Panel: `Cmd+Shift+P` -> "Agent Panel"

### 4. Добавить скиллы

Скиллы - это типовые запросы, которые AI выполняет по шаблону. Тексты лежат в `03. Ресурсы/Скиллы/`.

1. Открыть Rules Library: `Cmd+Alt+L`
2. Создать новое правило
3. Скопировать текст из файла скилла
4. Сохранить

Подробная инструкция: `03. Ресурсы/Скиллы/_как-добавить-скиллы.md`

### 5. Начать работу

Попробуйте вызвать скилл в Agent Panel:
- `@недельное-планирование` - спланировать неделю
- `@личный-бренд` - написать пост
- `@здоровье` - проверить состояние

## Структура

```
00. Планирование/     <- Ежедневные заметки, планы
01. Проекты/          <- Проекты с целью и сроком
02. Области/          <- Постоянные области ответственности
    Маркетинг/        <- Личный бренд + продукты/услуги
    Клиенты/          <- CRM: профили и заметки по сессиям
    Здоровье/         <- Профилактика выгорания
03. Ресурсы/          <- Шаблоны, методологии, скиллы
04. Архив/            <- Завершенные проекты
ZZ. Служебное/        <- Техническое (состояние AI, логи)
```

## Три уровня AI-работы

- **Просто хранение** - Архив. AI не трогает.
- **Хранение + анализ** - Проекты, CRM клиентов, Методологии. AI читает и анализирует по запросу.
- **С процессами** - Планирование, Маркетинг, Здоровье. AI запускает скиллы из Rules Library.

## Конфиденциальность

- Не вводите реальные имена клиентов - используйте псевдонимы
- API-ключ хранится в системной связке ключей, не в файлах
- Данные хранилища остаются на вашем компьютере
- AI-провайдер получает только текст, который вы отправляете в чат
